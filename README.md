# Plataforma de Ciberseguridad Municipal

## Descripción
Plataforma web y móvil desarrollada para municipios de Chile, enfocada en fortalecer la ciberseguridad institucional y proteger datos personales. Proyecto alineado con la **Ley 21.663 (Marco de Ciberseguridad)** y la **Ley 21.180 (Transformación Digital)**.

##  Guía de Instalación y Ejecución

### Herramientas necesarias (con las versiones ocupamos)

En nuestro proyecto ocupamos las siguietes herramientas para el desarrollo Front end :
- React
- ionic ( 7.2.1)
- nodejs (24.24.1)
- Lenguajes de programación como typescript
- Consolas windows y linux para ejecutar el proyecto
- Visual studio code (1.119.0)

### Instalacion:
Para instalar el proyecto debemos seguir las siguientes instrucciones:

1. Ingresar a [https://github.com/eliana-garcia/Proyecto-web-y-movil]

2. Llegaremos a algo como esto :
<img width="948" height="313" alt="imagen" src="https://github.com/user-attachments/assets/81a4d622-c981-420b-b55a-df04f6849fe5" />
En esta sección debemos pinchar en el boton que dice "code" y luego en download zip
<br>
<br>
 3. Una vez descargado se extrae el .zip y abrimos la carpeta y deberiamos encontrar lo siguiente :
<img width="711" height="460" alt="imagen" src="https://github.com/user-attachments/assets/27770db2-de34-4d45-8205-73b41d101d88" />

Entrar a la carpeta codigoProyecto y encontraremos casi todo los archivos necesarios para ejecucion , pero nos falta descargar la carpeta "node_modules" que contiene archivos necesarios.

 4. Para descargar los archivos debemos entrar a la terminal de linux o windows, al entrar nos aparecera algo como esto :
     <img width="815" height="559" alt="imagen" src="https://github.com/user-attachments/assets/785d25df-d0e9-46d0-9981-e524f8bb8685" />

    Una vez ahi debemos colocar el comando "cd + ruta" para acceder desde la terminal a la carpeta donde se encuentran nuestros archivos del frontEnd , osea la carpeta codigoProyecto
    nuestra ruta , sera lo siguiente:
    <img width="371" height="45" alt="imagen" src="https://github.com/user-attachments/assets/db951794-300d-420f-bbc9-4960bf0b6082" />

    Esto se encuentra en la aprte superior de nuestros administrador de archivo y lo que debemos es copiar toda la ruta , y colocarla junto con el "cd" y presionar enter.

 5. Una vez realizado eso nuestra terminal deberia cambiar a lo siguiente:
<img width="819" height="571" alt="imagen" src="https://github.com/user-attachments/assets/cc7bc4ba-97ec-4ea1-ab30-95ace96ce2d4" />
Una vez en esa seccion debemos escribir el comando "npm install" este comando accederá al package .json para descargar la carpeta "node_modules" y finalizar la instalacion. ahora se puede cerrar terminal. 

### Ejecución del proyecto y mostrar las paginas 
Para ejecutar el proyecto en el navegador debemos hacer lo siguiente:
1. Realizamos el paso 4 de la sección anterior para poder acceder a la carpeta donde se encuentran los archivos

2. Una vez ahi escribimos el siguiente comando "ionic serve"
   
3. Una vez ejecutado nos debería abrir el navegador y mostrar los siguiente:
